Request scope—Spring creates an instance of the bean for each HTTP request.

Session scope—Spring creates an instance of the bean per HTTP session of the
client. Multiple requests from the same client can share the same instance.

Application scope—There’s only one instance for the whole application for that
specific bean. Every request from any client can access this instance.
